Conway’s Game of Life – Portfolio Version
Description

Ce projet est une implémentation interactive du Jeu de la Vie de Conway en Python.
Il permet de simuler l’évolution d’une population d’êtres humains sur une grille en respectant les règles classiques :

Une cellule vivante meurt si elle a moins de 2 ou plus de 3 voisins vivants.

Une cellule morte devient vivante si elle a exactement 3 voisins vivants.

Le jeu propose une interface graphique avec possibilité d’interagir directement avec les cellules.

Fonctionnalités

Cliquer sur une cellule pour la rendre vivante ou morte.

Démarrer / arrêter la simulation avec la barre espace.

Avancer d’une génération avec la flèche droite.

Réinitialiser le plateau pour commencer une nouvelle simulation.

Affichage dynamique de la génération courante et de la population vivante.

Technologies utilisées

Python 3

Tkinter pour l’interface graphique

Concepts utilisés : classes, héritage, deepcopy, événements clavier et souris

Installation

Cloner le dépôt :

git clone https://github.com/rokhayan886-rgb/jeu-turmite.git

Se déplacer dans le dossier du projet :

cd jeu-turmite

Lancer le jeu :

python3 conway.py

Assurez-vous que S08_TP15.py (ou les modules nécessaires) est présent dans le même dossier.

Usage

Cliquer sur une cellule pour l’activer ou la désactiver.

Appuyer sur espace pour démarrer ou arrêter la simulation.

Appuyer sur flèche droite pour avancer d’une génération manuellement.

Cliquer sur reset pour réinitialiser la grille.
